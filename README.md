## 🎯 About

A stunning, fully responsive portfolio website showcasing my journey as an aspiring Software Development Engineer. Built with modern web technologies and featuring cutting-edge animations and 3D effects.

## ✨ Features

- 🎨 **Modern UI/UX** - Sleek glassmorphism design with gradient accents
- 🌀 **3D Animations** - Interactive geometric sphere and rotating skills cube
- 📱 **Fully Responsive** - Optimized for all devices and screen sizes
- 🎭 **Smooth Animations** - AOS (Animate On Scroll) library integration
- 🗣️ **Text-to-Speech** - Voice narration for About section
- 💫 **Interactive Elements** - Hover effects and smooth transitions
- 📥 **Downloadable CV** - One-click resume download
- 🔗 **Social Integration** - Direct links to GitHub and LinkedIn

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with animations
- **JavaScript** - Interactive functionality
- **AOS Library** - Scroll animations
- **Google Fonts** - Space Grotesk & Poppins

## 🎨 Design Highlights

### 🌟 3D Geometric Sphere

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS/JS (for customization)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/naveenkumar7449/My-Portfolio.git
   ```

2. **Navigate to the project**
   ```bash
   cd My-Portfolio
   ```

3. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use Live Server in VS Code
   ```

## 🔧 Customization

### Update Personal Information
Edit `index.html`:
- Name, tagline, and bio
- Social media links
- Contact details
- Project information

### Modify Colors
Edit `style.css`:
- Update gradient colors
- Change accent colors
- Modify background effects

### Add Projects
Add new project cards in the Projects section of `index.html`
